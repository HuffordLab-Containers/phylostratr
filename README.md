# phylostratr
docker container for phylostratr R version
Includes packages required to run PhylostratR

```r
library(devtools)
library(phylostratr)
library(reshape2)
library(taxizedb)
library(dplyr)
library(readr)
library(magrittr)
library(knitr)
library(ggtree)
library(phytools)
```
 
Also includes `blast` and `diamond` as well.
